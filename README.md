# Smart Spice LIB Editor

Python과 Tkinter로 만든 Smart Spice `.lib` 모델 파일 편집 GUI 도구입니다.

---

## 주요 기능

- **트리 기반 탐색** — `.LIB` 블록, `.MODEL` 항목, `.PARAM` 변수, 기타 지시어(`.directives`)를 사이드바 트리로 한눈에 파악
- **인라인 편집** — 파라미터 셀, 라이브러리/모델 이름을 트리에서 더블클릭하여 즉시 이름과 값을 수정
- **일괄 수정** — 특정 파라미터의 값을 현재 LIB 또는 전체 LIB 내의 모든 모델에서 한 번에 일괄 변경
- **파라미터 중심 뷰** — 파라미터별로 각 모델이 가진 값을 한눈에 비교할 수 있는 별도 뷰 제공
- **Excel 내보내기** — 전체 구조를 행렬/리스트 형태의 Excel 파일로 내보내어 손쉬운 외부 데이터 분석 지원
- **구문 마크업 완벽 보존** — 모델 파라미터 주변의 괄호(`()`), 원본 주석(`*`), 임의의 기타 지시어(`.temp` 등)를 파싱하여 형식 손상 없이 그대로 저장
- **내용 미리보기** — 저장 전 Smart Spice 문법으로 출력될 내용을 팝업으로 확인 (의미 단위 줄바꿈 유지)

```

```

## 요구 사항

- Python 3.8 이상
- Tkinter (Python 표준 라이브러리에 포함)
- `openpyxl` (Excel 내보내기 기능을 위해 필요)

> 엑셀 기능을 사용하기 위해 `pip install openpyxl` 명령어로 라이브러리를 설치해 주세요.

## 실행 방법

```bash
cd tr-fitting-manual
python3 main.py
```

실행 후 툴바의 **📂 파일 열기** 버튼을 클릭하여 `.lib` 파일을 불러옵니다.

---

## 사용법 요약

| 동작 | 방법 |
|---|---|
| `.lib` 파일 열기 | 툴바의 **📂 파일 열기** 클릭 |
| 이름 및 타입 수정 | 좌측 트리에서 `📁 LIB` 또는 `⚙️ MODEL` 더블클릭 |
| 모델 파라미터 조회 | 좌측 트리에서 `⚙️ MODEL` 항목 클릭 |
| 셀 값 및 이름 수정 | 우측 테이블 셀 더블클릭 → 텍스트 수정 후 `Enter` |
| 파라미터 추가/삭제 | 하단 **＋ 파라미터 추가** 또는 **－ 삭제** 클릭 |
| 일괄 수정 | 하단 **📋 일괄 수정** 클릭 후 대상 파라미터 지정 |
| 파라미터 중심 뷰 | 트리 하단 **🔄 파라미터 중심 뷰 열기** 클릭 |
| 내용 미리보기 | 툴바의 **👁 내용 미리보기** 클릭 |
| Excel 내보내기 | 툴바의 **📊 Excel 내보내기** 클릭 |
| 저장 | **💾 저장** (원본 덮어쓰기) 또는 **💾 다른 이름으로 저장** |

---

## 값 색상 코딩

| 색상 | 의미 |
|---|---|
| 🟢 초록 | 숫자 리터럴 (예: `0.45`, `1.2e-8`) |
| 🟡 노랑 | 변수 단순 참조 (예: `{tox_global}`) |
| 🟣 보라 | 산술 수식 (예: `{vth_offset + 0.40}`) |

## 출력 파일 형식

편집 후 저장하면 Smart Spice 문법을 유지하며, 긴 줄은 `+` continuation line으로 자동 줄바꿈(80자 기준)됩니다.

```spice
.PARAM tox_global=1.2e-8

.LIB NMOS_LIB
.PARAM vth_offset=0.05
.MODEL NMOS_V1 NMOS
+ LEVEL=3 VTH0=0.45 TOX={tox_global} XJ=1.5e-7
+ K1=0.559 K2=-0.04
.ENDL NMOS_LIB
```
