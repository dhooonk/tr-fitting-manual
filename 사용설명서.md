## 4. 트리뷰 탐색

- **📁 LIB: 이름** — `.LIB` 블록 (클릭 시 해당 블록 정보 표시)
- **📄 설정 (.directives)** — 해당 영역에 소속되는 일반 설정 지시어들(`.temp` 등)
- **🔧 PARAMS** — `.PARAM` 변수 목록 편집
- **⚙️ MODEL명 [타입]** — 특정 모델 선택 → 우측에 파라미터 목록 표시
- **📄 전역 설정** — `.LIB` 바깥에 위치한 전역 기타 지시어 목록
- **🔧 전역 PARAMS** — 파일 전체에 적용되는 전역 `.PARAM` 변수

> 항목을 클릭하면 우측 편집 패널이 해당 내용으로 전환됩니다.

## 5. 파라미터 편집

### 이름 및 속성 변경
1. 좌측 트리에서 변경하려는 **📁 LIB** 텍스트 구역이나 **⚙️ MODEL** 노드를 **더블클릭**
2. 팝업되는 대화상자에서 새 이름 또는 타입을 입력 후 **확인**

### 셀 값 직접 수정
1. 우측 테이블에서 수정할 파라미터(이름 혹은 값) 셀을 **더블클릭**
2. 값 입력 후 **Enter** 또는 **Tab** → 저장 ( **Esc** → 취소 )

### 일괄 수정
1. 하단 **`📋 일괄 수정`** 버튼 클릭
2. 대상 파라미터 이름과 새 값을 지정, 적용 범위(전체 또는 현재 LIB) 설정 후 **적용**
3. 일치하는 모든 모델에서 값이 일괄 교체됩니다.

### 파라미터 추가 / 삭제
- 추가: 트리에서 **MODEL** 이나 **PARAMS** 선택 후 **`＋ 파라미터 추가`** 
- 삭제: 우측 테이블에서 행을 클릭하고 **`－ 삭제`** 클릭

### 파라미터 중심 뷰 확인
- 트리의 하단이나 아래 기능에서 **`🔄 파라미터 중심 뷰 열기`** 버튼 클릭
- 특정 파라미터를 좌측 창에서 선택 시 우측에 전체 LIB/MODEL에서의 값 목록 표시

## 6. 변수(PARAM) 사용법

파라미터 값에 고정 숫자 대신 **변수**를 사용하면, 나중에 변수 하나만 바꿔도 여러 모델에 일괄 반영됩니다.

### 변수 설정 방법

1. 좌측 트리에서 **🔧 전역 PARAMS** 노드를 클릭하고 **`＋ 파라미터 추가`** 를 눌러 새 변수명(`tox_val`)과 값(`1.2e-8`)을 만듭니다.
2. 각 모델의 파라미터 편집창에서, 해당 변수가 들어갈 값 란을 더블클릭하여 `{tox_val}` 형태로 직접 적어줍니다.
3. 시뮬레이션 시 변수값이 치환되어 적용됩니다.

### 수식 예시

| 수식 표현 | 의미 |
|----------|------|
| `{tox_val}` | 변수 값 그대로 사용 |
| `{tox_val * 1.1}` | 변수 값 × 1.1 |
| `{vth_offset + 0.40}` | 변수 값 + 0.40 |
| `{K1 / 2}` | 변수 값 ÷ 2 |

> 수식은 Smart Spice가 시뮬레이션 시 자동으로 계산합니다.

### PARAMS 노드에서 직접 변수 추가/수정
1. 트리에서 **🔧 PARAMS** 선택
2. 우측 테이블에 변수 목록 표시됨
3. 더블클릭으로 변수명/값 수정, `＋` 버튼으로 신규 추가, `－` 로 삭제

## 8. 저장된 파일 형식 (Smart Spice 문법)

편집 후 저장하면 아래와 같은 형식으로 출력됩니다:

```spice
.PARAM tox_global=1.2e-8

.LIB NMOS_LIB
.PARAM vth_offset=0.05
.MODEL NMOS_V1 NMOS
+ LEVEL=3 VTH0=0.45 TOX={tox_global} XJ=1.5e-7
+ K1=0.559 K2=-0.04
.ENDL NMOS_LIB
```

- 긴 줄은 `+` continuation line으로 자동 줄바꿈 (80자 기준)
- 원래 있던 `#` 또는 `*` 주석 줄과 빈 줄, `.temp` 같은 기타 `.지시어`의 위치 및 내용이 그대로 유지됩니다.
- 모델 내용 파싱/저장 시 파라미터를 묶는 `( )` 괄호 구조 또한 원형대로 완벽하게 보존됩니다.
